# Hi, I'm Rosemary 👋

I build **ML systems for biological data** - production-grade pipelines, models, and platforms at the intersection of machine learning and bioinformatics.

My background is in ML Platform Engineering. I care about shipping models that actually work in production: reproducible, monitored, and scalable.

---

### 🧬 What I Work On

- Machine learning on complex, high-dimensional data - with a focus on biological and clinical datasets
- Production ML pipelines - from raw data to deployed model
- MLOps infrastructure: reproducible, monitored, scalable
- APIs and platforms that make models actually usable
